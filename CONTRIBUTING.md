## Thanks for visiting! If you are thinking to contribute, go through the guidelines once.

You can contribute us by filing issues, bugs and PRs. You can also take a look at active issues and fix them.

If you want to discuss on something, or some feature then feel free to discuss it on [discussions](https://github.com/nav0804/Sentinel-Guard/discussions). 

### Code contribution

- Open issue regarding proposed change.
- If your proposed change is approved, Fork this repo and do changes.
- Open PR against latest *dev* branch. 
- Add nice description in PR.
- That's it!!

### Code contribution checklist

- New code addition/deletion should not break existing flow.
- Verify `turbo run build` is passing before raising a PR.
- Reformat code before raising a PR.
